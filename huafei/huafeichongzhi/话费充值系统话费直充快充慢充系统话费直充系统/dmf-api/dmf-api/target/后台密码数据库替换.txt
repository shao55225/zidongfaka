如果都不对用以下方式修改：
打开源代码目录中的登录逻辑文件，一般是login.php或者authenticate.php。

定位到用户登录时密码加密的部分，该部分通常包含如下代码：

$password = $_POST['password'];
$hashed_password = hash($hash_algorithm, $password);
其中，$hash_algorithm表示当前使用的加密算法。

将$hash_algorithm改为’md5’，如下所示：

$password = $_POST['password'];
$hashed_password = md5($password);
这样就使用了MD5算法对密码进行了加密处理。

保存修改后的源代码文件并重新部署应用程序，然后使用新的密码进行登录测试，确保修改生效。

注意：MD5算法已经不被建议使用。更安全的方式是使用加盐的哈希算法，例如bcrypt或scrypt等。
